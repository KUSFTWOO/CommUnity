### [15:00] render.yaml 파일 생성
```yaml
# 웹 서비스 (community-web)
- type: web
  runtime: python
  buildCommand:
    - pip install -r requirements.txt
    - python manage.py collectstatic --noinput
    - python manage.py migrate
  startCommand: gunicorn config.wsgi:application --bind 0.0.0.0:$PORT
  
# 환경 변수 자동 연결
- DATABASE_URL: community-db 자동 연결
- SECRET_KEY: 임의값 생성

# PostgreSQL 데이터베이스
- type: pserv (managed database service)
- plan: free
- region: ohio
```

### [15:15] 로컬 서버 테스트
