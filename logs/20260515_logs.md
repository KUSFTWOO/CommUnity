### 14:45 — CustomUser 모델 작성

**accounts/models.py:**

```python
class CustomUser(AbstractUser):
    email = models.EmailField(unique=True)  # 중요: unique 필수
    nickname = models.CharField(max_length=30, unique=True)
    bio = models.CharField(max_length=150, blank=True)
    profile_image = models.ImageField(upload_to='profiles/%Y/%m/', blank=True, null=True)
    
    login_fail_count = models.PositiveSmallIntegerField(default=0)
    locked_until = models.DateTimeField(null=True, blank=True)
    is_deleted = models.BooleanField(default=False)
    
    USERNAME_FIELD = 'email'
    REQUIRED_FIELDS = ['username', 'nickname']
```

**메서드 추가:**
- `is_account_locked()` — 계정 잠금 확인
- `unlock_account()` — 계정 해제
- `increment_login_fail()` — 실패 횟수 증가
- `reset_login_fail()` — 실패 횟수 초기화
