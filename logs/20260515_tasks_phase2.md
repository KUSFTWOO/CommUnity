  - 사용자명 필드
  - 닉네임 필드 (중복 검사)
  - 비밀번호 필드 (8자 이상, 일치 검증)
  - Tailwind CSS 스타일링
- ✅ `signup()` 뷰
  - 기존 사용자 로그인 시 홈으로 리다이렉트
  - 폼 유효성 검사
  - 사용자 생성 & 자동 로그인
  - 성공 메시지 표시

  - 카드 형식 디자인

- ✅ `LoginForm` 폼 작성
  - 이메일 필드
  - 비밀번호 필드
  - 로그인 실패 처리
    - 계정 잠금 확인 (30분)
    - 실패 카운팅 (5회 제한)
    - 카운트다운 메시지
- ✅ `login_view()` 뷰
  - 기존 사용자 로그인 시 홈으로 리다이렉트
  - 폼 유효성 검사
  - 로그인 성공 시 실패 카운트 초기화
  - 사용자명 포함 환영 메시지
- ✅ `login.html` 템플릿
  - 카드 형식 디자인
  - 회원가입 링크
  - 비밀번호 찾기 링크
  - 테스트 계정 정보 표시

### 3. 로그아웃 (Logout)
- ✅ `logout_view()` 뷰
  - Django 기본 logout 사용
  - 성공 메시지 표시
  - 홈으로 리다이렉트

  - 프로필 이미지 표시

  - 프로필 이미지 업로드

- ✅ `profile_edit()` 뷰

  - 이미지 미리보기
