## 📌 기술 메모

**CustomUser 보안 메서드:**
```python
- is_account_locked()      # 계정 잠금 확인
- unlock_account()         # 계정 해제
- increment_login_fail()   # 실패 카운팅
- reset_login_fail()       # 초기화
```

**폼 검증 레벨:**
1. 필드 레벨 (Field validation)
   - clean_email(), clean_password()
2. 폼 레벨 (Form validation)
   - clean()

**로그인 흐름:**
```
POST /accounts/login/
  ↓
LoginForm.clean() 호출
  ├─ 이메일 존재 여부 확인
  ├─ 계정 잠금 여부 확인
  ├─ 비밀번호 검증
  └─ 성공 시 reset_login_fail()
  ↓
login(request, user) 호출
  ↓
세션 생성 & 쿠키 설정
  ↓
홈으로 리다이렉트
```
